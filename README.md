## 🎯 Objetivo do Projeto

Conforme atividade da DIO, estou utilizando o **NotebookLM** como ferramenta de aprendizado. O objetivo principal é estruturar, estudar e fixar o máximo de conhecimento sobre conceitos fundamentais de banco de dados para quem está dando os primeiros passos na área de tecnologia.
